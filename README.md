# JarvisV2

Its a updated version of AI assistant JARVIS from iron man movies.

=================== LAUNCH =============================
To launch progect and run it in your device your should:
1. Copy repository to your folder
2. Create virtual enviroment or just install packages in your python using pip install -r requirements.txt
3. Create your API keys for services and add them in .env file
4. That all now u can use JARVIS as u want.
=========================================================




=====================NEW FEACHURES=======================
1. Jarvis now using vosk model for offline and faster speach recognision
2. Added some new functions like screenshot, recent news, now u can say what music you want to listen and Jarvis will turn it on.
3. Added keyboard interupt if you dont want to say goodbye you can press 'q' and u will leave the command listener section and Jarvis will turn to wake for command state.
